Información general sobre el OVI

El presente trabajo comprende el estudio de lenguaje de Diseño web HTML5, Estructura básica de HTML5, Aplicación de elementos de HTML5 a un proyecto de diseño Web, Introducción a CSS3, Estructura básica de CSS3, Aplicación de elementos de CSS3 a un proyecto de diseño Web.

Objetivos del OVI

* Diseñar Sitios Web que satisfagan las necesidades del usuario

* Operar lenguaje de Diseño web como HTML5

* Operar lenguaje de Diseño web como CSS3
